<로그인 기능>
아이디/비번 입력
	> 아이디/비번 틀렸을 경우 + 로그인한 계정이 블랙리스트일 경우
	   로그인 X

	> 아이디/비번 맞았을 경우
	   로그인	

로그인한 아이디의 회원정보를 가져옴
	> 회원정보의 mCheck가 Y
		> 관리자 메뉴 출력
	> 회원정보의 mCheck가 N
		> 일반회원 메뉴 출력

<회원가입>
아이디 입력
MEMBERS 테이블에서 같은 아이디 사용중인지 확인
	> 같은 아이디가 있는경우
	   	> 아이디 재입력
	
	> 같은 아이디가 없는 경우
	  	> 회원가입 진행
			>비밀번호,이름,생년월일,이메일 입력
				>MEMBERS에 INSERT

<관리자메뉴>
1. 상품등록
등록할 상품 이름 검색
	> 동일 상품 없을 시
		> 등록절차 실시
	> 동일 상품 있을 시
		> 해당 상품이 있다는 출력문

2. 상품수정
수정할 상품이름 입력
	> 해당 상품이 있을시
		>상품정보 출력
		>수정할 정보 선택 (수량 & 가격)
	> 없을 시 
		>없다는 출력문

3. 회원관리
메뉴출력(1. 회원목록 / 2.회원검색 / 3.회원권한 )
	-회원목록
		> db에서 MEMBERS 가져오기
			> 출력
	-회원검색
		>검색할 회원 아이디 입력
			> 등록된 회원일 시 회원정보 출력
			> 없을 시 "등록되지 않은 회원" 출력
	- 회원권한
		> 권한바꿀 회원 아이디 입력
			> 아이디 있으면 회원정보 저장
				> 메뉴출력(1.관리자 2. 블랙 3.일반)
					>  mCheck 변경


<일반회원메뉴>
1. 충전
로그인한 계정 정보를 받음
	> 충전할 금액 입력
		> ???금액을 충전할것인지 확인
			> 맞다면 MEMBERS 에 업데이트
				> 로그인계정 정보 다시 불러오기
			> 아니라면 충전취소
			
				> 현재 소지금 출력

2. 상품목록
상품카테고리 선택
	> 카테고리에 맞는 상품목록 출력


3. 상품검색
검색할 상품명 입력
입력된 내용이 들어간 상품명 SELECT
데이터가 있는경우 출력


4. 주문하기
상품명 입력
수량 입력
구매하기 / 장바구니에 담기 선택

구매하기
	> 총 가격 구하기
	> 소지금 >= 총가격 && 상품수량 >= 구매수량
		> ORDERLIST에 UPDATE
		> UPDATE 성공시
			> MEMBERS의 login회원 MCASH,MTOTAL,MGRADE 업데이트
			> loginInfo 다시 불러오기

			> PRODUCT의 해당상품 수량 감소

장바구니 담기
	> 로그인 ID 장바구니 내역 검색
	> 해당 상품이 있음
		> 수량추가
	> 해당 상품이 없음
		> CART TABLE에 INSERT



5. 장바구니 내역
로그인 ID의 CART리스트 출력

1. 구매결정
	> 구매할 상품의 번호 입력
	> 해당 상품 정보 가져오기
	> 해당 상품 구매
		> ORDERLIST INSERT
		> PRODUCT UPDATE
		> MEMBERS UPDATE
		> CART DELETE

	> 로그인정보 다시 불러오기
	> 장바구니 리스트 다시 불러오기

2. 장바구니삭제
	> 삭제할 상품번호 입력
	> CART DELETE


6. 내정보
내정보 보기
	> 회원정보 출력

구매취소
	> 구매리스트 출력
	> 구매취소할 상품 번호 입력
	> 해당상품 정보 받기
		> ORDERLIST DELETE
		> PRODUCT AMOUNT UPDATE
		> MEMBERS CASH/GRADE/TOTAL UPDATE
